Related network tools

Ping — measure round-trip latency and packet loss.
Traceroute — see every hop between you and a host.
Whois — lookup domain registration and ownership.
NSLookup — resolve A, AAAA, MX and other DNS records.
Nmap — scan for open ports and services on a host.
SNMP Walk — query SNMP OIDs over the network.
SIP — test if a SIP server is reachable and responding.
Penetration Test — discover hidden directories on a web server.
People Search — find social profiles across networks.
Port Check — test if a TCP port is open on a host.
Reverse DNS — PTR lookup for any IP.
Subnet Calculator — CIDR, netmask and host ranges.
What Is My IP — see your public IP, ISP and location.
User Agent — decode your browser, OS and device.
MAC Lookup — find the manufacturer of a MAC address.
HTTP Status — follow any URL's redirect chain.
Is It Down? — check if a website is online.
SSL Checker — inspect TLS certificate and chain.
