About this People Search

People Search queries the public web — and the social graphs of Facebook, Instagram, Twitter and TikTok — for a person's name or handles in a single step. It is useful for reconnecting with someone, verifying identity before a hire, or checking how much of your own digital footprint is public.

Fill in any combination of full name and handles. The more fields, the narrower the result. Cards arrive in real time as each query completes; the cover image of each page is fetched asynchronously and filled in on each card.

Tips

  • Use the full name in quotes to avoid mixed results with similar names — type "Jane Doe".
  • Handles work best with the @. If you're unsure, drop the @; we add it for Instagram/Twitter.
  • Not finding someone? They may have a private profile. Public-only search won't reveal them by design.

FAQ

How does it work? We query a public search index and filter results by site (facebook.com, instagram.com, twitter.com, tiktok.com). No login, no scraping of private data.

Is it legal? Yes, we only surface publicly available content. Respect local privacy laws when acting on the information.

Why are some cards missing images? Some pages don't expose an Open Graph image, or the site blocks our server from downloading it.
